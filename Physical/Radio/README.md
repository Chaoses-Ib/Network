> 通过观察发现，大部分英特尔无线网卡主板只有数字1，2，3，其中标识的MAIN数字为1（连灰线），AUX数字为2（连黑线），如果有3，就连白线。  
> 但是到了后期有些M.2接口的网卡，MAIN数字为2和AUX数字为1。
